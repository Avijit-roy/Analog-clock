# Analog Clock

An elegant and functional analog clock built with modern web technologies. Perfect for keeping track of time in style with a classic, timeless design.

## Features

- **Real-Time Display**: Shows accurate time synchronized with your system clock
- **Elegant Design**: Clean and attractive analog clock interface
- **Smooth Animation**: Fluid movement of hour, minute, and second hands
- **Responsive Layout**: Works seamlessly on desktop and mobile devices
- **Lightweight**: Fast loading and minimal resource consumption

## Demo

Try the live clock here: [https://avijit-roy.github.io/Analog-clock/index.html](https://avijit-roy.github.io/Analog-clock/index.html)

## Technologies Used

- **HTML**: Clock structure and markup (index.html)
- **CSS**: Styling and visual design (anaclock2.css)
- **JavaScript**: Real-time functionality and hand movement (anaclock3.js)

## Installation

No installation required! Visit the [live demo link](https://avijit-roy.github.io/Analog-clock/index.html) to view the clock immediately.

To run locally:
1. Clone the repository
2. Open `index.html` in your web browser
3. The clock will automatically display the current time

## How It Works

The analog clock uses JavaScript to:
- Fetch the current system time
- Calculate the rotation angles for the hour, minute, and second hands
- Update the display in real-time
- Create smooth, continuous hand movement

## Browser Compatibility

Works on all modern browsers including:
- Chrome
- Firefox

- Edge
